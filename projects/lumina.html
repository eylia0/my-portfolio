Lumina Challenge — Seat of Collaboration

An interactive stool designed for Sydney VIVID — collaboration becomes play, and light becomes the reward.

Experience Public Interaction Sydney VIVID

Project intent

Lumina turns teamwork into a physical action. The stool separates into portable parts, prompting strangers to cooperate through a simple challenge structure. When the form is correctly reassembled, illumination confirms completion — a visible “yes”.

Core idea

  • Parts invite participation
  • Assembly requires cooperation
  • Light rewards the shared outcome

Process

From interaction concept → assembly logic → material tests → final illuminated prototype.

Ideation

Exploring interactive formats for VIVID — shifting from “stool first” to “interaction first”.

Idea pivot → interaction drives form.

Material + pattern tests

Testing “brick” layout, spacing, and translucency — balancing structure with light bleed.

Pattern testing → structure + glow.

Assembly logic

Developing a no-fastener interlock system that can be understood quickly in public.

No fasteners → intuitive assembly.

Final prototype

Refining edges, stability, and lighting placement so the final result reads as “complete”.

Finish + light → confirmation moment.

Outcome

A minimal object with an unusual behaviour: it becomes whole through people — not instructions.
